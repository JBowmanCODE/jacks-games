# Jack's Games

A collection of browser games built with Next.js.

**🎮 Play live: [https://jacks-games-ten.vercel.app](https://jacks-games-ten.vercel.app)**

## Games

- Chess
- Drop Four (Connect 4)
- Football
- Hangman
- Ocean Battle (Battleship)
- Paintball 5v5
- Snakes
- Spelling
- Sudoku
- Wordle
- Wrestling

## Run locally

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).
